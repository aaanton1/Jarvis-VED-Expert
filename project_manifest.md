# JARVIS — Project Manifest

## Цель
Конвейер валидации бизнес-идей и управления AI-агентами.
Система автономно исследует рынки, собирает данные и предоставляет аналитику через единый интерфейс.

## Стек технологий
| Слой       | Технология              |
|------------|------------------------|
| Frontend   | Next.js (App Router, TypeScript) |
| UI         | Tailwind CSS           |
| Backend    | Node.js                |
| Database   | SQLite                 |

## Структура проекта
```
/frontend   — UI-интерфейс (Next.js)
/backend    — Логика агентов-исследователей
/data       — JSON-результаты поиска
/docs       — Документация и "память" проекта
/logs       — Логи работы системы
```

## Правила разработки
1. Каждая новая фича фиксируется в `/docs/features.md`
2. Ошибки агентов пишутся в `/logs/agent_errors.log`
3. Перед крупным шагом создаётся `plan.md` и согласовывается с владельцем
4. Архитектурные решения документируются в `/docs/research_log.md`
