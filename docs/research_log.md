## 2026-02-15 | Инициализация проекта

**Решение:** Развёрнут фундамент проекта JARVIS.
- Выбран стек: Next.js + Node.js + SQLite + Tailwind
- Создана структура директорий
- Frontend инициализирован с App Router и TypeScript

**Обоснование:** SQLite выбран для MVP — легковесный, не требует отдельного сервера. При масштабировании можно мигрировать на PostgreSQL через ORM (Prisma/Drizzle).
