# Plan: Import Navigator — Модуль интервью + отчёт по импорту

> Дата: 2026-02-18
> Товар-пилот: Школьные пеналы (текстиль + пластик/EVA, дети до 14 лет)

---

## Цель

Создать `scripts/import_navigator.ts` — CLI-инструмент, который:
1. Проводит интервью с селлером (5-7 вопросов)
2. Автоматически определяет код(ы) ТН ВЭД
3. Генерирует полный **Import Brief** — отчёт со всеми данными для растаможки

---

## Шаг 1: Новые коды ТН ВЭД в ved-knowledge.ts

Добавить 2 кода для пеналов:

| Код | Описание | Пошлина | НДС | Сертификация |
|-----|----------|---------|-----|-------------|
| `4202320000` | Пеналы, футляры из текстиля (с молнией) | 10% | 20% | ТР ТС 007/2011 (дети) |
| `3926909700` | Пеналы, футляры из пластмасс (EVA, полипропилен) | 6.5% | 20% | ТР ТС 007/2011 (дети) |

Оба кода:
- `requiresCertification: true`
- `certTypes: ["Сертификат соответствия ТР ТС 007/2011 (безопасность детской продукции)"]`
- `requiresMarking: false` (пеналы не подлежат маркировке Честный ЗНАК)
- `riskNote` — предупреждение о переквалификации детских товаров

FTS пороги: текстильные $0.50, пластик $0.30.

## Шаг 3: scripts/import_navigator.ts

### Архитектура
```
import_navigator.ts
  ├── interview()        → задаёт 5-7 вопросов через readline
  ├── classifyProduct()  → ищет коды в TN_VED_DATABASE по keywords + материалу
  ├── generateBrief()    → компилирует Import Brief
  └── main()             → CLI entry point
```

### Import Brief (выходной отчёт)

```
╔═══════════════════════════════════════╗
║   IMPORT BRIEF — Школьные пеналы     ║
╚═══════════════════════════════════════╝

📦 ТОВАР
  Школьные пеналы (микс: текстиль + пластик/EVA)
  Аудитория: дети до 14 лет
  Застёжка: молния | Электроника: нет | Лицензия: нет

📊 ПАРТИЯ
  Кол-во: 500 шт
  Цена: $0.30/шт → $150.00 итого
  Страна: Китай | Логистика: карго

🏷 КОД ТН ВЭД
  ┌─ Текстильные: 4202320000 — Пеналы из текстиля
  │  Пошлина: 10% | НДС: 20% | Акциз: 0%
  │
  └─ Пластиковые: 3926909700 — Пеналы из пластмасс
     Пошлина: 6.5% | НДС: 20% | Акциз: 0%

💰 РАСЧЁТ ПЛАТЕЖЕЙ (пример: 250 текстиль + 250 пластик)
  Текстиль:  250 × $0.30 = $75 → пошлина $7.50 → НДС $16.50 → итого $24.00
  Пластик:   250 × $0.30 = $75 → пошлина $4.88 → НДС $15.98 → итого $20.85
  ВСЕГО таможенных платежей: ~$44.85

📋 СЕРТИФИКАЦИЯ
  ⚠️ ОБЯЗАТЕЛЬНЫЙ СЕРТИФИКАТ (не декларация!)
  ТР ТС 007/2011 — Безопасность продукции для детей и подростков
  • Протокол испытаний в аккредитованной лаборатории
  • Тест на фталаты, формальдегид, миграция красителей
  • Стоимость: ~15,000–30,000 руб за 1 сертификат
  • Срок: 2-4 недели

⚠️ РИСКИ
  1. ЗАНИЖЕНИЕ — $0.30/шт ниже порога ФТС ($0.50 для текстиля)
     → высокий риск КТС (корректировка таможенной стоимости)
     → Рекомендация: запросить у поставщика export invoice, packing list,
       контракт с полной спецификацией
  2. Детская продукция — повышенное внимание ФТС
     → обязателен именно СЕРТИФИКАТ, не декларация
  3. Карго без ДТ — если сумма < $200, карго-оператор может
     провести без полного декларирования, но для WB нужен
     полный пакет документов (сертификат + ДТ)

✅ ЧЕКЛИСТ ДОКУМЕНТОВ
  □ Контракт с поставщиком (на англ/кит)
  □ Инвойс (commercial invoice)
  □ Упаковочный лист (packing list)
  □ Сертификат соответствия ТР ТС 007/2011
  □ Протокол испытаний
  □ Транспортная накладная (от карго)
  □ Платёжное поручение (оплата поставщику)
```
